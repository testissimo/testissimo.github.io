Getting Started

Creating a Repo for our tests

To create your first Remote Repo click on Repos and click the + button.
After naming your first repository click on the Create New Repo button.
To access and use this Repo you now need to Download (Checkout) it.

In the next section we are going to create our first test with Testissimo.

Creating your first Test

Click on the newly created Repo to open a new tab to manage your tests and test suites.
In the new tab select Tests > Create New Test and name it as you wish and click Create Test.

In the next section we are going to write our very first test.

Writing your first test with Testissimo

In our first test we will try to discover HTML text inputs.

Then we will fill them up with strings and after that press a Submit button.

Follow these steps to complete this section

  1. Open your recently created Test
  2. In order to select a HTML element write select and click on Pick
  3. Click on the Name input text-box
  4. Testissimo has now selected this element and you can decide what to do with it
  5. Write do and then select write to simulate text input
  6. Selecting the Test me button is very similar to step 2 - write select and then click on Pick
  7. Click on the Test me button
  8. Write click to simulate a mouse click
  9. To check the result we need to select this element and assert its text value. Select this element and write assert. A combo-box will appear and in this combo-box you should select element(s) text is
  10. Run the test by clicking the Play button in the top-right corner. If the test finishes without any errors your test is correct and the form works well, too

Here is a sandbox form where you can try Testissimo out

Your test code should look like this

select: #nameInput
do write: David

select: input[type="submit"]:value("Test me")
do click

select: #result
assert element(s) text is: David
