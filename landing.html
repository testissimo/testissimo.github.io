Setting Up Testissimo

3 steps from writing your first Test

Download Add-On

If you are using Google Chrome or Mozilla Firefox, you can simply Install the plugin from the Add-ons library. Alternatively, you can insert a short piece of code into your web application (index.html)

Add-on for

or use the code below

<script id="testissimo" src="https://app.testissimo.io/testissimo.min.js"><script>

Sign Up

When Testissimo plugin is installed, you can simply activate it and fill the form for registration.

Your nick will be a prefix to all of your test repositories.

Start Writing Tests

After filling all the information, you are now ready to start using Testissimo.

We advise you to go through the documentation, if you are using Testissimo for the first time.
